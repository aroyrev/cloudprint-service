NAME

cps-auth − Perform OAuth2 authentication for cloudprint-service

SYNOPSIS

cps-auth [<user name>]

DESCRIPTION

The cps-auth provide a sudo wrapper for entering the required OAuth2 credentials for cloudprint-service. It will launch a browser window to complete the transaction, request a code from the browser session, and then store the credentials locally for cloudprint-service.

See cloudprint-auth for an alternative mechanism, if a graphical desktop is not available on the cloudprint-service server.

FILES

/var/lib/cloudprint/authfile

Credential storage location
