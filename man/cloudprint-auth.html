NAME

cloudprint-auth − acquire OAuth2 credentials remotely for cloudprint-service

SYNOPSIS

cloudprint-auth [<user name>]

DESCRIPTION

The cloudprint-auth performs the OAuth2 authentication for cloudprint or cloudprint-service, and outputs the credentials to stdout, in the format for the cloudprint authentication file. It is intended for remote use when the cloudprint-service server does not have a graphical environment.

cloudprint authentication files are stored by default in ~/.cloudprintauth. For cloudprint-service, the file is at /var/lib/cloudprint/authfile.
