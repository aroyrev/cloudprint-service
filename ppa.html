Using the cloudprint-debian repository.

Here is how to add cloudprint to apt repository resources for your system, letting you install and automatically upgrade with standard tools.

First, add a reference to the cloudprint PPA to your /etc/apt/sources.list file (as root):

deb http://davesteele.github.io/cloudprint-debian/repo <dist> main
... where <dist> is replaced with the appropriate value from this list:

  • jessie
  • testing
  • wheezy
  • stable
  • sid
  • unstable
  • saucy
  • raring
Add the repository key to your apt key ring:

wget http://davesteele.github.io/key-366150CE.pub.txt
sudo apt-key add key-366150CE.pub.txt
Install the packages with :

sudo apt-get update
sudo apt-get install cloudprint cloudprint-service
The packages will be updated with every 'apt-get upgrade'.
